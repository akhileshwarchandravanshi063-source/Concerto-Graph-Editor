# Concerto Graphical Editor

A web-based graphical editor for creating and editing Concerto models visually, without writing `.cto` schema code manually.

## 🚀 Project Overview

Currently, Concerto models are created using text-based syntax, which can be difficult for beginners to understand.
This project aims to simplify the process by providing a visual interface where users can design models using nodes and connections.

## ✨ Features Implemented

* Add nodes representing model entities
* Connect nodes to define relationships
* Interactive graph-based UI using React Flow
* Basic visual modeling of schema structures

## 🧠 Future Goals

* Export graph → Concerto `.cto` format
* Import `.cto` → visual graph
* Support advanced features:

  * Inheritance
  * Enumerations
  * Relationships
  * Arrays & Maps
* Integration with Template Playground

## 🛠️ Tech Stack

* React.js
* React Flow
* JavaScript
* HTML/CSS

## ⚙️ Installation & Setup

```bash
git clone https://github.com/akhileshwarchandravanshi063-source/Concerto-Graph-Editor.git
cd Concerto-Graph-Editor
npm install
npm start
```

Then open:
http://localhost:3000
